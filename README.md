# TFM – Aplicación de Realidad Virtual para Rehabilitación con CLAP

## Contexto

La rehabilitación de las extremidades superiores (manos y brazos) es clave para recuperar la autonomía en la vida diaria, especialmente en pacientes con lesiones neurológicas o problemas de movilidad.

En los últimos años, la realidad virtual ha emergido como una herramienta prometedora en este ámbito, permitiendo crear entornos inmersivos donde los pacientes pueden realizar ejercicios de forma más motivadora y controlada.

## El proyecto

Este Trabajo de Fin de Máster consiste en el desarrollo de una aplicación inmersiva en realidad virtual orientada a la rehabilitación de las extremidades superiores.

El objetivo principal es:

- Evaluar la movilidad del paciente
- Facilitar ejercicios de rehabilitación
- Registrar y analizar su progreso

Todo ello mediante una experiencia natural basada en la manipulación de objetos virtuales.

## Cómo funciona

La aplicación ha sido desarrollada en **Unity** e integra la tecnología **CLAP**, que permite una interacción más realista e intuitiva con las manos en entornos virtuales.

El sistema propone diferentes pruebas dentro de un entorno simulado (una cocina), donde el usuario realiza acciones cotidianas como:

- Mover objetos
- Girar utensilios
- Beber de un vaso
- Abrir y cerrar grifos o electrodomésticos

Estas tareas están diseñadas para trabajar habilidades motoras reales de forma natural.

## Características clave

- Rehabilitación basada en tareas reales (actividades de la vida diaria)
- Interacción natural con las manos gracias a CLAP (Utiliza hand tracking)
- Sistema de evaluación mediante tiempo y puntuación
- Seguimiento del progreso con gráficos de resultados
- Gestión de usuarios y almacenamiento de datos

La aplicación permite registrar cada intento del usuario y analizar su evolución a lo largo del tiempo.

## Resultados

El sistema no solo guía al usuario durante las pruebas, sino que también genera métricas que permiten evaluar el rendimiento, comparar intentos y visualizar la evolución del paciente. Esto aporta valor tanto para el propio usuario como para profesionales de la rehabilitación.
